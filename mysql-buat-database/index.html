Mysql : Buat Database Mysql di Terminal

#database

Di postingan sebelumnya kita sudah mengetahui apa itu mysql dan cara installnya, kali ini kita akan membuat database mysql di terminal,

dan database mysql sendiri bisa dibuat dengan phpmyadmin lewat browser untuk cara installnya kalian bisa cari tutorialnya sendiri. Baik kalau gitu mari kita mulai saja buat databasenya :

  • Masuk ke dalam pengolahan data mysql dengan memasukkan perintah berikut di terminal anda
sudo mysql -u root -p

Kemudian masukkan password anda yg sudah di setinng sebelumnya jika berhasil akan muncul tampilan seperti berikut

  • Setelah masuk kita bisa buat database dengan memasukkan perintah
CREATE DATABASE db_name;

Ganti db_name dengan nama database yg ingin kita buat, saya akan membuat database dengan nama belajar oh iya setelah kita mengetikkan perintah di mysql jangan lupa kasih tanda ; (titik koma)

CREATE DATABASE belajar;

jika berhasik akan ada keterangan Query ok

Dan untuk mengecek database kita, bisa memasukkan perintah

SHOW DATABASE;

Maka akan tampil semua database yg sudah pernah kita buat

Dan untuk perpindah ke halaman database kita bisa mengetikkan perintah

USE db_name;

Tadi kita buat database dengan nama belajar, maka masukkan perintah

USE belajar;

Maka akan tampil seperti ini

Sampai saat ini kita sudah bisa membuat database mysql dengan nama belajar, setelah ini kita bisa membuat table di dalam database belajar. Untuk membuat table kita bisa memasukkan perintah

CREATE TABLE nama_table (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(255),
    alamat VARCHAR(255)
);

Ganti nama_table dengan nama tabel yg ingin kalian buat, kali ini saya kan membuat nama table dengan nama siswa

CREATE TABLE siswa(
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(255),
    alamat VARCHAR(255)
);

Jika berhasil akan tampil seperti berikut

Untuk memasukkan data ke dalam table siswa kita bisa memasukkan perintah

INSERT INTO siswa(nama, alamat) VALUES ("Irham", "Bungu");

Ganti data sesuai keinginan anda, dan jika berhasil akan ada keterangan Query ok

Untuk melihat data di dalam table siswa kita bisa memasukkan perintah

SELECT * FROM siswa;

Untuk siswa ganti sesuai dg nama table anda. Maka tampil seperti berikut

  • Untuk menghapus data pada table bisa memasukkan perinta
DELET FROM siswa WHERE id=1;

Itu untuk menghapus data berdasarkan id nya.

Dan untuk Menghapus database bisa memasukkan perintah

DROP DATABASE db_name;
